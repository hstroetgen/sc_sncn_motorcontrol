qei Interface API¶

Client API¶

{unsigned int, unsigned int} g get_qei_position(chanend c_qei, qei_par &qei_params)¶

Get position from QEI Server.

Parameters:
  • c_qei – for communicating with the QEI Server
  • qei_params – the struct defines sensor type and resolution parameters for qei
Returns:

position from qei sensor in the range [0 - log(encoder_resolution)/log(2)]

valid for qei with index sensors: not valid - 0/ valid - 1

{int, int} g get_qei_position_absolute(chanend c_qei)¶

Get absolute position from QEI Server.

Parameters:
  • c_qei – for communicating with the QEI Server
Returns:

counted up position from qei sensor (incorporates set max ticks) in the range [ -max ticks to +max ticks]

direction of rotation, clockwise : 1 / anti-clockwise : -1

int get_qei_velocity(chanend c_qei, qei_par &qei_params, qei_velocity_par &qei_velocity_params)¶

Calculates the velocity from QEI sensor in 1 ms loop.

Parameters:
  • c_qei – for communicating with the QEI Server
  • qei_params – the struct defines sensor type and resolution parameters for qei
  • qei_velocity_params – struct for velocity calculation
Returns:

velocity from qei sensor in rpm

Server API¶

void run_qei(chanend c_qei_p1, chanend c_qei_p2, chanend c_qei_p3, chanend c_qei_p4, chanend c_qei_p5, chanend c_qei_p6, port in p_qei, qei_par &qei_params)¶

Implementation of the QEI server thread (for sensor with index/no index).

Parameters:
  • c_qei_p1 – the control channel for reading qei position in order of priority (highest) 1 ... (lowest) 5
  • c_qei_p2 – the control channel for reading qei position priority - 2
  • c_qei_p3 – the control channel for reading qei position priority - 3
  • c_qei_p4 – the control channel for reading qei position priority - 4
  • c_qei_p5 – the control channel for reading qei position priority - 5
  • c_qei_p6 – the control channel for reading qei position priority - 6
  • p_qei – the hardware port where the quadrature encoder is located
  • qei_params – the struct defines sensor type and resolution parameters for qei
