Client API¶

int get_hall_position(chanend c_hall)¶

Gets position from Hall Server.

Parameters:
  • c_hall – A chanend connected to Hall Server
Returns:

the position in the range [0 - 4095] which maps to [0 - 359]/pole-pairs
{int , int} g get_hall_position_absolute(chanend c_hall)¶

Gets absolute position from Hall Server.

Parameters:
  • c_hall – A chanend connected to Hall Server
Returns:

the counted up position (compensates for pole-pairs) in the range [0 - 4095] * pole-pairs

Server API¶

void run_hall(chanend c_hall_p1, chanend c_hall_p2, chanend c_hall_p3, chanend c_hall_p4, chanend c_hall_p5, chanend c_hall_p6, port in p_hall, hall_par &hall_params)¶

A basic hall sensor server.

Parameters:
  • c_hall_p1 – the control channel for reading hall position in order of priority (highest) 1 ... (lowest) 6
  • c_hall_p2 – the control channel for reading hall position (priority 2)
  • c_hall_p3 – the control channel for reading hall position (priority 3)
  • c_hall_p4 – the control channel for reading hall position (priority 4)
  • c_hall_p5 – the control channel for reading hall position (priority 5)
  • c_hall_p6 – the control channel for reading hall position (priority 6)
  • p_hall – the port for reading the hall sensor data
  • hall_params – struct defines the pole-pairs and gear ratio
