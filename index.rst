SOMANET Motor Control Library
==============================

.. toctree::
	:maxdepth: 1
	:hidden:

	Module Motorcontrol <module_motorcontrol/doc/index>
	Module Hall Sensor Interface <module_hall/doc/index>
	Module Encoder Interface <module_qei/doc/index>

The **SOMANET Motor Control Library** provides Motor Control support for SOMANET devices. 

Our SOMANET Motor Control Library is composed of the following modules:

* `Module Motorcontrol <module_motorcontrol/doc/index>`_
* `Module Hall Sensor Interface <module_hall/doc/index>`_
* `Module Encoder Interface <module_qei/doc/index>`_
* Module ADC.
* Module Blocks.
* Module Control Loops.
* Module GPIO.
* Module Motorcontrol Common.
* Module Profile.
* Module watchdog.
